# Pure Function Lab

## 实验目标
验证纯函数的核心特性：无副作用和引用透明性

## 验收标准(AC)

### AC1: 无副作用验证
- [ ] 验证纯函数不修改输入参数
- [ ] 验证纯函数不修改外部状态
- [ ] 验证纯函数不进行I/O操作

### AC2: 引用透明性验证  
- [ ] 验证相同输入总是产生相同输出
- [ ] 验证函数调用可被返回值替换
- [ ] 验证等式推理能力

### AC3: 纯函数优势验证
- [ ] 验证可测试性（无需mock）
- [ ] 验证可缓存性（memoization）
- [ ] 验证并行安全性

## 实验步骤

1. 编写纯函数和非纯函数示例
2. 验证无副作用特性
3. 验证引用透明性
4. 演示纯函数的实际优势
