# Stream API Lab

## 实验目标
验证Stream API的核心概念，包括惰性求值、操作管道和并行处理

## 验收标准(AC)

### AC1: 流操作管道验证
- [ ] 验证中间操作和终止操作的区别
- [ ] 验证惰性求值特性
- [ ] 验证操作顺序的重要性

### AC2: 常用流操作验证
- [ ] map/filter/reduce操作链
- [ ] 集合收集操作(Collectors)
- [ ] 排序和去重操作

### AC3: 并行流验证
- [ ] 验证并行流的使用场景
- [ ] 验证线程安全性要求
- [ ] 验证性能对比

### AC4: 高级特性验证
- [ ] 无限流(Stream.iterate/generate)
- [ ] 原始类型流(IntStream等)
- [ ] 自定义收集器

## 实验步骤

1. 创建基本流操作示例
2. 验证惰性求值特性
3. 实现并行流处理
4. 演示高级流特性
